### 猫屋App (Cat House App)
##### 该项目是我在深圳大学软件工程专业核心课《软件工程综合实训》中的作业，最终成绩接近满分（97），并名列专业第一。
#### 项目概述
随着社会对流浪动物的关注日益增加，猫咪救助与爱猫互动的需求逐渐成为社区关注的焦点。猫屋App旨在提供一个爱猫社区平台，让用户通过关注猫窝、投喂猫粮、直博互动等功能，不仅为流浪猫提供救助支持，还促进了猫咪爱好者之间的互动与交流。

#### 技术栈
- **前端**: Android, XML, Java
- **后端**: Spring Boot
- **数据库**: MySQL, MyBatis 3
- **认证**: Sa-Token
- **版本控制**: Git
